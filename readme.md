# Verificare se una parola è palindroma

- Inserire nell'HTML un input dove permettere all'utente di inserire una parola
- Recuperare dal DOM l'elemento input e dove stampare il risultato
- Creare una funzione
  - valore sempre falso
  - **SE** la parola inserita dall'utente è uguale alla parola girata al contrario
  - impostare valore vero
  - **ALTRIMENTI** printare valore falso
- Stampare in pagina
  - Non è palindroma
  - **SE** la funzione da risultato vero, 'La parola è palindroma'

# Pari e dispari

- Creo un input dove l'utente sceglie pari o dispari e inserisce un valore da 1 a 5
- Richiamo l'input e la scelta paro-disparo dal DOM
- Creo una funzione che generi un numero casuale da 1 a 5 (min a max)
- CPU genera numero random
- Sum i due numeri
- Con una funzione applichiamo un modulo sul numero del risultato per dichiarare se il numero è paro
  - **SE** il numero da resto 0 è _true_
  - **ALTRIMENTI** è _false_
- **SE** l'utente seleziona _pari_ e il modulo da risultato _true_
  - l'utente ha vinto
- **ALTRIMENTI**
  - l'utente ha perso
